# TenderHackRiskAnalysis
Модель для анализа рисков при заключении договора.

### Датасет

Датасет состоит из абзацов для классификации и таргетов.
Таргеты принимают следующие значения:
- Ценовые риски
- Срок оплаты
- Аванс
- Налоговые сборы
- Изменение цены
- Пени
- Просрочка
- Штраф
- Нет риска или риск не известен
